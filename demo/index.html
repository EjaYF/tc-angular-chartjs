tc-angular-chartjs demo

Line Chart

Using tc-chartjs directive

<canvas tc-chartjs chart-type="line" chart-data="data" chart-options="options" width="350" height="350"></canvas>

Using tc-chartjs-line directive

<canvas tc-chartjs-line chart-data="data" chart-options="options" width="350" height="350"></canvas>
